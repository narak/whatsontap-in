WhatsOnTap.in /bengaluru

What is it?
A portal that shows you WhatsOnTap around you. Want a quick pint before heading back home from work? Where is the best IPA in town? Where is the closest Irish Red Ale I can get? When was Toit Tintin last tapped? What is the word on the new Mango Pale Ale at Arbor Brewing Co.?

Wouldn't it be nice to have all of this this? Well, its probably very difficult but I've felt the need for something like this for a while. I'm working on the website for it first since that is my professional expertise. The logistics of getting the content is something I haven't thought about yet.
If you're interested in this, feel free to drop me a line at whatsup[a t]whatsontap.in.

Features

The darker it is, the higher priority it is.
The lighter it is, the lower priority it is.

Brewery List

  • Beers on tap
    • Beer details (description, type, IBU, ABV, price)
    • Expert reviews
    • User reviews
  • Beers coming up
    • Beer details
  • Brewers login
    So brewers can login themselves and update all the values. Initially, there will be no incentive for the brewers to login and add the info themselves, so we'd have to do it ourselves. So this gets bumped down in priority.

Bottled beers

  • Where to find them (which wine shops stock them)
  • Expert reviews
  • User reviews
  • Wine shop login (low priority, low incentive)
